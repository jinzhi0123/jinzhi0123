PicGo+Github+jsDelivr搭建个人图库

发表于2021-08-31|更新于2021-10-24|博客搭建
|字数总计:374|阅读时长:1分钟|阅读量:

1.PicGo安装

首先进入PicGo官网下载安装包

点击免费下载进入GitHub，选择对应系统的安装包

进行安装

完成安装

2.Github配置

在Github创建一个仓库，名称随意

建立一个Token

依次点击 Settings -> DeveloperSettings -> Personal access tokens -> Generate new token

勾选repo权限并创建

复制得到的Token

3.PicGo配置

  • 仓库名：Github用户名/图床仓库名
  • 分支名：仓库的分支名
  • Token：之前生成的Token
  • 存储路径：存储图片的路径，默认为 img/
  • 自定义域名：使用CDN加速访问GitHub，这里使用jsDelivr,格式为 https://cdn.jsdelivr.net/gh/用户名/仓库名

4.关于jsDelivr

jsDelivr是一个为npm、github等提供免费CDN的项目，使用其可以在全球范围内为你的项目提供访问加速服务，更多的用法可见jsDelivr官网。

jsDeliver无需配置，只需要格式符合要求，即可加速访问。比如，https://cdn.jsdelivr.net/gh/jinzhi0123/image/img/test.png 即为访问jinzhi0123这个GitHub账户下的image仓库中img目录下的test.png文件。

5.PicGo与Typora配合使用

Typora设置

Typora是一款非常常用的markdown文本编辑器，PicGo与Typora搭配使用非常舒服。

如图配置，选择PicGo(app)并设置安装路径。之后在Typora插入图片时就会自动上传并将本地路径改为网络路径。

你也可以在PicGo的相册中已上传的图片以md的格式直接复制到Typora。
